## 🚀 How to Use

1. **Open the portfolio in a browser:**
   - Simply open `index.html` in your web browser
   - No server or build process required

2. **Customize the content:**
   - Edit `index.html` to update text, contact info, and projects
   - Modify `styles.css` to change colors, fonts, or animations
   - Update `script.js` for additional functionality

3. **Add your projects:**
   - Replace placeholder images in the portfolio section
   - Update project titles, descriptions, and tags
   - Modify the `data-category` attribute for filtering

4. **Personalize the design:**
   - Update the color scheme in CSS variables (`:root`)
   - Change fonts in the body styles
   - Adjust animation timing and effects

## 🎨 Customization Guide

### Colors
Edit the CSS variables at the top of `styles.css`:
```css
:root {
    --primary-color: #9b59b6;      /* Purple */
    --secondary-color: #3498db;    /* Blue */
    --accent-color: #e74c3c;       /* Red */
    --dark-bg: #1a1a2e;            /* Dark background */
    --light-bg: #fff;              /* Light background */
}
```

### Animation Speed
Adjust animation duration in CSS:
- Change `0.8s` to faster/slower speeds
- Modify `@keyframes` to customize effects
- Update `transition` properties

### Content
- Update personal information in HTML
- Change project details and images
- Modify skills and expertise areas
- Update contact information
